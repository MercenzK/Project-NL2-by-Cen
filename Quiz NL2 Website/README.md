# NL2 Med Quiz — เว็บทำข้อสอบออนไลน์

เว็บทำข้อสอบ NL2 (อายุรศาสตร์) 500 ข้อ พร้อมระบบล็อกอินและบันทึกคะแนนรายคน
ทำงานได้ทันทีแบบ **Guest mode** (เก็บคะแนนในเครื่อง) และเปิด **ล็อกอิน + คะแนนข้ามเครื่อง** ได้ด้วย Supabase (ฟรี)

## ไฟล์ในโฟลเดอร์นี้
- `index.html` — ตัวเว็บ (เปิดไฟล์นี้ได้เลย)
- `data.js` — ข้อสอบ 500 ข้อ (5 ชุด)
- `config.js` — ที่ใส่ค่า Supabase (แก้เฉพาะไฟล์นี้)
- `supabase_schema.sql` — สคริปต์สร้างตารางเก็บคะแนน
- `README.md` — คู่มือนี้
- `test_app.js` — ไฟล์ทดสอบ ลบทิ้งได้

---

## วิธีลองเปิดเร็ว ๆ (บนเครื่องตัวเอง)
ดับเบิลคลิก `index.html` เปิดในเบราว์เซอร์ก็ใช้ได้เลย (Guest mode) — ทำข้อสอบ เฉลย นับคะแนน เก็บประวัติในเครื่องนี้

> หมายเหตุ: การล็อกอินต้องต่ออินเทอร์เน็ต (โหลด Supabase จาก CDN)

## เอาขึ้นออนไลน์ฟรี — Netlify (ง่ายสุด ไม่ต้องใช้ Git)
1. ไปที่ https://app.netlify.com/drop
2. ลาก **ทั้งโฟลเดอร์** `Quiz NL2 Website` วางลงในหน้าเว็บ
3. รอสักครู่ จะได้ลิงก์ (เช่น `https://ชื่อสุ่ม.netlify.app`) — เสร็จ! แชร์ลิงก์ได้เลย

อัปเดตข้อสอบ/แก้ไฟล์ ก็ลากโฟลเดอร์วางใหม่ทับได้

### หรือ GitHub Pages
1. สร้าง repo ใหม่บน GitHub แล้วอัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้
2. Settings → Pages → Branch: `main`, folder `/root` → Save
3. รอสักครู่ จะได้ลิงก์ `https://ชื่อผู้ใช้.github.io/ชื่อ-repo/`

## เปิดระบบล็อกอิน + คะแนนรายคน (Supabase ฟรี)
ทำครั้งเดียว ~5–10 นาที

1. สมัคร/เข้า https://supabase.com → **New project** (ตั้งชื่อ + รหัสผ่าน database, เลือก region Singapore)
2. รอโปรเจกต์สร้างเสร็จ → เมนู **SQL Editor** → New query → วางเนื้อหาไฟล์ `supabase_schema.sql` ทั้งหมด → **Run**
3. เมนู **Project Settings → API** จะเห็น
   - **Project URL** (เช่น `https://abcxyz.supabase.co`)
   - **anon public** key (ขึ้นต้น `eyJ...`)
4. เปิด `config.js` เอา 2 ค่านี้ใส่:
   ```js
   SUPABASE_URL: "https://abcxyz.supabase.co",
   SUPABASE_ANON_KEY: "eyJhbGciOi....",
   ```
5. (แนะนำ) เมนู **Authentication → Providers → Email** เปิด Email ไว้
   - อยากให้สมัครแล้วใช้ได้เลยไม่ต้องยืนยันอีเมล: ปิด "Confirm email"
6. เอาไฟล์ขึ้น Netlify/GitHub ใหม่ (ตามด้านบน) — ทีนี้ปุ่ม "เข้าสู่ระบบ" จะใช้ได้ สมัคร/ล็อกอินด้วยอีเมล+รหัสผ่าน คะแนนจะผูกกับบัญชีและตามข้ามเครื่องได้

> ความปลอดภัย: anon key ใส่ในเว็บได้ตามปกติ เพราะข้อมูลถูกป้องกันด้วย Row Level Security (แต่ละคนเห็นเฉพาะคะแนนตัวเอง)

## อัปโหลดชุดข้อสอบเพิ่มเอง
หน้าแรก → ปุ่ม **⬆️ อัปโหลดชุดข้อสอบ** → เลือกไฟล์ `.json` หรือ `.csv` (เก็บในเบราว์เซอร์เครื่องนี้)

### รูปแบบ JSON
```json
{
  "title": "ชุดของฉัน",
  "questions": [
    {
      "topic": "Cardiology",
      "q": "โจทย์...",
      "choices": ["ตัวเลือก A","B","C","D","E"],
      "ans": 0,
      "exp": "คำอธิบายเฉลย..."
    }
  ]
}
```
- `ans` = ลำดับตัวเลือกที่ถูก (เริ่มที่ 0 = A) หรือใส่เป็นตัวอักษร `"A"` ก็ได้
- มี 2–5 ตัวเลือกก็ได้

### รูปแบบ CSV (แถวแรกเป็นหัวตาราง)
```
topic,question,A,B,C,D,E,answer,explanation
Cardiology,โจทย์...,ตัวเลือกA,B,C,D,E,A,คำอธิบาย...
```
- `answer` ใส่เป็นตัวอักษร (A–E) หรือเลข (0–4)

## ข้อจำกัด / หมายเหตุ
- ข้อสอบเป็น "แนวข้อสอบ" เฉลยเรียบเรียงอ้างอิงแนวทางล่าสุด — ตอนสอบจริงยึดแนวทางไทยเป็นหลัก
- ชุดข้อสอบที่อัปโหลดเองจะเก็บใน localStorage ของเบราว์เซอร์ (เฉพาะเครื่อง/เบราว์เซอร์นั้น)
- ถ้าไม่ตั้งค่า Supabase เว็บยังใช้ได้เต็มรูปแบบ เพียงแต่คะแนนเก็บในเครื่องและไม่มีล็อกอิน
